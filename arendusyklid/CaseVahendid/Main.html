Case vahendid

Case vahendid on tööriistad, mida kasutatakse tarkvaraarenduses süsteemide planeerimiseks, modelleerimiseks ja arendamiseks.

Mille jaoks neid kasutatakse?

Case vahendid aitavad automatiseerida arendusprotsessi, parandada dokumentatsiooni ja tagada arenduse kvaliteeti.

Kuidas jaotatakse case vahendeid?

Case vahendid jagunevad Uppercase ja Lowercase tööriistadeks.

Milliseid vahendeid saab kasutada Agile arendusmudeli elutsüklis?

  • Planeerimine: Jira
  • Disain: draw.io
  • Arendus: Visual Studio Code
  • Testimine: Selenium
  • Deployment: Docker

Milliseid vahendeid olen ise juba kasutanud?

  • draw.io
  • Visual Studio Code
  • Docker
